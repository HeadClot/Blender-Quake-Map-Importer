# Blender-Quake-Map-Importer
A Blender 5.1 Quake 1 Map Importer that imports Q1 Valve 220 format maps into blender. Supports textures in PNG format as well.


**How to Import a Quake 1 Map:**

Install the addon as you noramlly would. 
Go to file - Import and look for "Quake Map"
Select a .map file that is in the Valve 220 format
Select your Target Engine (Unreal, Unity, Godot, or Custom)
Press "Import Quake Map"

**How to apply textures on to your Quake 1 Map.**

After you import your Quake 1 Map. Go to your N Panel.
You should see a New tab called "Quake Map".

Click on that tab click the folder icon. That will open a file browser. 
Navigate to where you have you textures in .PNG format and go into the folder then hit accept.
Hit apply textures after. Your textures should now be applied to your Imported map file.

**Limitations**:

This only supports Valve 220 format quake 1 maps.
This only supports .PNGs for textures.

Important Notes: 

I recommend to use Trenchbroom and then Select the Generic profile + Valve 220 format.
Then load your PNGs into your textures folder.
